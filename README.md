# tapebox
A quantion box built in Vuejs.

If you want to build your own box, you need to modify tape url in App.vue.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
